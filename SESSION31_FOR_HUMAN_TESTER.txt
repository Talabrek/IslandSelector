=================================================
HOW IT WORKS
=================================================

1. Player runs: /islandselector slots
2. Slot Selection GUI opens
3. Player RIGHT-CLICKS on slot 1
4. Click type detected (RIGHT vs LEFT)
5. System checks:
   - Is slot locked? ‚Üí Error
   - Is slot empty? ‚Üí Error
   - Has island? ‚Üí Open settings
6. SlotSettingsGUI opens
7. Shows:
   - Slot preview (center)
   - Rename button (left)
   - Change Icon button (middle)
   - Delete button (right)
   - Back button (bottom)
8. Player clicks a button
9. GUI closes, command instructions shown
10. Player uses command to make changes

Example Flow:
- Right-click slot 1
- Settings GUI opens
- Click "Rename" button
- See: "Use command: /islandselector setname 1 <name>"
- Run: /islandselector setname 1 My Base
- Slot renamed!
