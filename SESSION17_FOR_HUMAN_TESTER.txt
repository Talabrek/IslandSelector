=================================================
IMPORTANT TESTING NOTES
=================================================

**Search uses CHAT INPUT:**
When you click the search button, the GUI closes and you type the
player name directly in chat. This is normal Minecraft plugin behavior.

**Your search input should NOT be visible to other players!**
The chat event is intercepted and cancelled, so your search input
is private.

**Partial matching is case-insensitive:**
- "not" finds "Notch"
- "test" finds "TestPlayer"
- "ste" finds "Steve"

=================================================
KNOWN LIMITATIONS (BY DESIGN)
=================================================

1. Chat-based input - This is a Minecraft limitation, no GUI text
   input is available in inventory GUIs

2. Prefix matching only - "Not" finds "Notch", but "otch" doesn't
   (this is by design for performance)

3. Single result - If multiple players have similar names, returns
   first match only

4. Offline search limited - Only finds offline players who have
   islands registered in the database
