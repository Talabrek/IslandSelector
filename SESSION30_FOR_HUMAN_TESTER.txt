=================================================
HOW IT WORKS
=================================================

1. Player runs: /islandselector delete 2
2. Command validates slot exists
3. Opens confirmation GUI with warnings
4. Player clicks "DELETE SLOT" (red button)
5. Logic determines:
   - If last slot ‚Üí delete island from world + grid
   - If active slot ‚Üí switch to another slot first
   - If non-active slot ‚Üí delete immediately
6. Slot removed from database
7. Success message displayed

Example (Last Slot):
- Player: /islandselector delete 1
- GUI: "‚ö† This is your only slot! Deleting it will remove your island from the grid completely."
- Player clicks: DELETE SLOT
- Result: Island removed from world, grid location freed, can claim new island

Example (Active Slot):
- Player: /islandselector delete 1 (while in slot 1)
- Warning: "‚ö† Warning: This is your currently active slot!"
- Player confirms
- Message: "This is your active slot. Finding another slot to switch to..."
- Message: "Switching to slot #2..."
- (5 second delay for switching)
- Message: "Slot #1 has been deleted!"
